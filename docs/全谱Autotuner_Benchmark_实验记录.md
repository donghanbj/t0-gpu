# 全谱 Autotuner Benchmark 实验记录

## 日期
2026-03-31

## 目标
验证 tile_ir autotuner 在 10 个不同矩阵尺寸下的稳定性和性能表现。

## 环境
- GPU: AMD RX 7900 XTX (GFX1100, 96 CU)
- BF16 WMMA GEMM, 理论峰值 ~165 TF
- KFD 裸金属运行时（无 HIP）

## 结果

| Size | TFLOPS | Best Config | 效率 (vs 165T) |
|------|--------|-------------|----------------|
| 256³ | 2.6 | 64x64_k64 | 1.6% |
| 512³ | 15.0 | 64x64_k64 | 9.1% |
| 1024³ | 51.9 | 64x64_k64 | 31.5% |
| 2048³ | 94.2 | 64x64_k64 | 57.1% |
| 4096³ | 103.7 | 64x128_k32 | 62.8% |
| **8192³** | **116.8** | **128x128_k64** | **70.8%** |
| 128×4096×1024 | 24.2 | 64x64_k64 | 14.7% |
| 256×4096×1024 | 47.2 | 64x64_k64 | 28.6% |
| 512×4096×1024 | 67.4 | 64x64_k64 | 40.8% |
| 1024×4096×1024 | 76.9 | 64x64_k32 | 46.6% |

## 关键发现

### 1. 尺寸-配置映射规律
- **≤1024³**: 64x64 tile 最优（128x128 生成的 WG 太少，CU 大量空闲）
- **2048³-4096³**: 中间地带，64x128/128x64 混合胜出
- **≥8192³**: 128x128 tile 最优（每个 WG 做更多工作，减少调度开销）
- **固定默认 64x64_k64 对于 ≤2048 是最佳选择**

### 2. k 深度规律
- **方形大矩阵**: k64 > k32（更高 WMMA 占比）
- **方形中矩阵(4096)**: k32 ≈ k64（VGPRs 更省）
- **非方形**: k64/k32 接近，k32 在 M 较大时有优势

### 3. 稳定性
- 10 个尺寸 × 12-13 候选 = ~120 次 GPU dispatch
- **零 hang，零 timeout**
- 总耗时 10.57s（含编译 + benchmark）
- Spill 过滤正常工作（64x128_k64 被跳过）
